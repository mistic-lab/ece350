Message Sink

Input Type
Don't Block
Vec Length

Creates messages from an incoming data stream. Message passing is the mechanism that allows data to flow between the Python and C++ layers of GNU Radio.

Input Type

Specifies the data type of the input stream.

Complex Input stream is complex.
Float Input stream is real.
Int Input stream is 32-bit integer.
Short Input stream is 16-bit integer.
Byte Input stream is 8-bit byte.

Don't Block

Type:

Leave as default.

Vec Length

Type: int

Specifies the vector length for vector processing. Typical applications will use the default value of 1.
