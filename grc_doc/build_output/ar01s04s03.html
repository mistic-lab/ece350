Parameter

Label
Value
Type
Short ID

This block represents a parameter to the flow graph. A parameter can be used to pass command line arguments into a top block. Or, parameters can pass arguments into an instantiated hierarchical block. The paramater value cannot depend on any variables. When type is not None, this parameter also becomes a command line option of the form: -[short_id] --[id] [value] The Short ID field may be left blank.

Label

Type: string

Leave the label blank to use the parameter id as the label.

Value

Type: Same as Type.

Sets the default value if not specified on the command line.

Type

Specfies the data type of the resulting variable.

None No type.
Complex Parameter is interpreted as complex.
Float Parameter is interpreted as real.
Int Parameter is interpreted as an integer.
Long Parameter is interpreted as a long integer.
String Parameter is interpreted as a string.

Short ID

Type: string

An optional short-hand form for the parameter. May be left blank.
